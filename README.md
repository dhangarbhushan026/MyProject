# 💰 Expense Tracker

A simple and smart **Expense Tracker** app that helps users to record, manage, and analyze their daily expenses easily.

---

## 📋 Features
- Add daily expenses with category and amount  
- View total spending summary  
- Edit or delete expense records  
- Simple and clean interface  

---

## 🛠️ Technologies Used
- **Python** (Backend)
- **HTML, CSS** (Frontend)
- **Flask** (To connect frontend and backend)

---

Follow these simple steps to run the project on your own computer 👇
